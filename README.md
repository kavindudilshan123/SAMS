# Student Attendance Management System (SAMS)

## Overview
This is a simple Student Attendance Management System developed using Java.

## Features
- User Login
- Add Students
- View Students
- Add Courses
- Add Lecturers
- Mark Attendance
- View Attendance

## Technologies Used
- Java
- MySQL

## How to Run
1. Open project in IDE (NetBeans / IntelliJ)
2. Run Main.java
